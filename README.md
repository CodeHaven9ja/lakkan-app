## Using this project

Install or Upadate last version Ionic Cli

```bash
$ npm install -g ionic
```

Install NPM Packages:

```bash
$ npm install
```

Start Ionic Serve:

```bash
$ ionic serve
```



More info on this can be found on the Ionic [Getting Started](http://ionicframework.com/docs/v2/getting-started/) page.
